# dva-crud

---

## Getting Started
Install dependencies

```bash
$ npm install
```

Start server
```bash
$ npm start
```

Install json-server
```bash
$ npm install -g json-server
```

Start mock server
```bash
$ npm run mock
```

If you do correct, application will start in http://localhost:8000/ and json-server will start in http://localhost:9090/
